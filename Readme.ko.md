# Wavoscope - 오디오 분석 및 전사 도구

Wavoscope는 음악가, 전사 전문가 및 오디오 엔지니어를 위해 설계된 강력한 실시간 오디오 시각화 및 전사 보조 도구입니다. 고정밀 파형, 스펙트럼 분석 및 견고한 마커 시스템을 제공하여 복잡한 오디오를 분석하는 데 도움을 줍니다.

![메인 인터페이스](docs/images/main_view.png)

---

## 🚀 시작하기

### Wavoscope 실행
Wavoscope는 자기 완결형으로 설계되었습니다. Python이나 기타 의존성을 수동으로 설치할 필요가 없습니다.
- **Windows:** `run.bat`를 더블 클릭합니다. 그러면 자동으로 환경이 설정되고 향후 사용을 위해 루트 폴더에 `Wavoscope.exe`가 생성됩니다.
- **Linux/macOS:** 터미널에서 `bash run.sh`를 실행합니다. 그러면 루트 폴더에 `Wavoscope` 바이너리가 생성됩니다.

처음 실행할 때 Wavoscope는 자체 Python 런타임을 자동으로 다운로드하고 필요한 환경을 설정합니다. 인터넷 연결 상태에 따라 몇 분 정도 걸릴 수 있습니다. 첫 실행 후에는 생성된 `Wavoscope` 실행 파일(앱 아이콘 포함)을 사용하여 간단히 실행할 수 있습니다.

### 프로젝트 관리 및 자동 저장
Wavoscope는 "사이드카" 파일 시스템을 사용합니다. 오디오 파일을 열면 Wavoscope는 동일한 디렉토리에 마커, 루프 및 설정을 저장하기 위해 `.oscope` 파일을 생성하거나 로드합니다.
- **열기:** 재생 바의 폴더 아이콘을 클릭하여 일반적인 오디오 형식(MP3, WAV, FLAC 등)을 로드합니다.
- **저장:** 플로피 디스크 아이콘을 클릭합니다. 저장되지 않은 변경 사항이 있으면 아이콘이 테마의 강조 색상으로 빛납니다.
- **자동 저장:** Wavoscope는 정기적으로 작업의 스냅샷을 자동으로 생성합니다. 자동 저장 빈도, 유지할 최대 스냅샷 수 및 저장 위치는 **설정 > 복구** 탭에서 구성할 수 있습니다. 기본적으로 자동 저장은 저장되지 않은 변경 사항이 있을 때만 발생합니다. 변경 여부와 관계없이 항상 스냅샷을 생성하도록 **강제 자동 저장**을 활성화할 수 있습니다. 기본적으로 자동 저장은 시스템의 임시 폴더에 저장됩니다.

## 🎵 탐색 및 재생

- **확대/축소:** 파형 또는 스펙트럼 위에서 **마우스 휠**을 사용하여 확대/축소합니다.
- **스크롤:** **타임라인** 위에서 **마우스 휠**을 사용하여 시간을 앞뒤로 스크롤합니다.
- **팬(Pan):** 파형 또는 스펙트럼을 **클릭하고 드래그**하여 타임라인을 이동합니다.
- **적응형 세분화:** 확대/축소에 따라 타임라인이 그리드 단계(0.01초부터 몇 시간까지)를 자동으로 조정하여 너무 조밀하지 않으면서 최적의 세부 정보를 보장합니다.
- **재생 커서:** 파형을 **왼쪽 클릭**하여 재생 헤드를 이동합니다.
- **속도 제어:** 하단 바의 슬라이더를 사용하여 속도를 0.1배에서 2.0배까지 조절합니다. Wavoscope는 피치를 유지하는 고품질 타임 스트레칭을 사용합니다.

## 🔍 스펙트럼 분석 및 필터링

화면 하단에는 피아노 롤에 매핑된 정항 Q 변환(CQT) 스펙트로그램이 표시됩니다. 스펙트럼 분석기 헤더의 컨트롤을 사용하여 **FFT 윈도우** 및 **옥타브 시프트**를 조정할 수 있습니다.

![스펙트럼 필터링](docs/images/spectrum_filter.png)

### 고급 필터링
실시간 대역 통과 필터를 사용하여 특정 악기나 음을 분리할 수 있습니다. 필터 핸들(스펙트럼의 수직선)은 항상 사용할 수 있습니다:
- **컷오프 토글:** 필터 핸들을 **오른쪽 클릭**하여 해당 경계를 활성화하거나 비활성화합니다.
- **빠른 배치:** 스펙트로그램의 아무 곳이나 **오른쪽 클릭**하여 가장 가까운 필터 핸들을 이동하고 활성화합니다.
- **시각적 피드백:** 컷오프가 활성화되면 범위 밖의 영역이 어둡게 표시되어 집중을 돕습니다. 둘 다 비활성화되면 필터가 바이패스됩니다.

## 🚩 플래그 및 전사

Wavoscope는 트랙의 구조와 화성을 매핑하는 데 도움이 되는 이중 플래그 시스템을 사용합니다.

### 리듬 플래그 (리듬/마디 마커)
- **배치:** `B`(기본값)를 누르거나 타임라인을 **왼쪽 클릭**하여 리듬 플래그를 놓습니다.
- **세분화:** 플래그 대화 상자(플래그 핸들 **오른쪽 클릭**)를 열어 세분화(예: 4분 음표의 경우 4)를 설정합니다. 타임라인에 희미한 수직선으로 표시됩니다.
- **메트로놈:** 세분화 클릭이 활성화된 경우 재생 중에 리듬 플래그가 자동으로 메트로놈 클릭을 트리거합니다.
- **Shift-클릭:** 이전 플래그와 동일한 간격으로 새 플래그를 자동으로 배치합니다. 일정한 비트를 빠르게 매핑하는 데 적합합니다.
- **섹션:** 플래그를 "섹션 시작"으로 표시하여 레이블("Verse" 또는 "Chorus" 등)을 지정할 수 있습니다.

![리듬 플래그 대화 상자](docs/images/rhythm_dialog.png)

### 코드 플래그 (코드 마커)
- **배치:** `C`(기본값)를 누르거나 타임라인을 **오른쪽 클릭**하여 코드 플래그를 놓습니다.
- **코드 편집기:** 기존 플래그를 **오른쪽 클릭**하여 코드 대화 상자를 엽니다. 코드 이름(예: "Am7", "C/G")을 입력하거나 선택기를 사용할 수 있습니다.
- **자동 분석:** **제안** 버튼을 사용하여 Wavoscope가 해당 위치의 오디오를 분석하고 가장 가능성 있는 코드를 추천하도록 합니다.
- **들어보기:** 코드 플래그 핸들을 **왼쪽 클릭 유지**하거나 대화 상자의 "재생" 버튼을 클릭하여 내부 합성기를 통해 연주되는 코드를 들을 수 있습니다.

![코드 플래그 대화 상자](docs/images/harmony_dialog.png)

### 플래그 관리
- **드래그:** 타임라인의 플래그 핸들을 **클릭하고 드래그**하여 위치를 미세 조정할 수 있습니다.
- **겹침:** 리듬 플래그와 코드 플래그가 동일한 공간을 차지할 경우, 두 플래그를 모두 조작할 수 있도록 절반 높이(코드는 위, 리듬은 아래)로 표시됩니다.
- **루핑:** 재생 바의 루프 버튼을 사용하여 마커 사이 또는 전체 트랙 루프를 전환합니다.

## 🎤 가사 전사

Wavoscope는 고속 전사 및 정렬을 가능하게 하는 대화형 가사 트랙을 제공합니다.

![가사 전사](docs/images/lyrics_track.png)

### 전사 워크플로우
1. **트랙 토글:** 파형 헤더의 "가사" 버튼을 클릭하여 전사 트랙을 표시합니다.
2. **추가 및 입력:** `V`를 누르거나 가사 트랙의 빈 공간을 **한 번 클릭**하여 단어를 추가합니다.
3. **고속 입력:** 가사 상자에 입력하는 동안 **스페이스** 또는 **대시(`-`)**를 누릅니다. 그러면 자동으로 다음이 수행됩니다:
    - 현재 단어를 확정합니다.
    - 즉시 뒤따르는 새 가사 상자를 생성합니다(현재 재생 헤드 또는 이전 끝 지점).
    - 음악을 멈추지 않고 계속 입력할 수 있도록 포커스를 새 상자로 이동합니다.
4. **이동:** `Shift + 왼쪽/오른쪽` 또는 `Shift + A/D`를 사용하여 가사 요소 사이를 점프합니다. 타이밍 확인에 적합합니다.

### 편집 및 크기 조절
- **이동:** 가사 상자의 중앙 80%를 **드래그**하여 이동합니다.
- **타이밍:** 가사 상자의 가장자리(10% 임계값)를 **드래그**하여 시작 또는 종료 시간을 조정합니다.
- **정밀도:** 가사가 선택되었을 때 **화살표 키**를 사용하여 0.1초 단위로 미세 조정합니다. **위/아래** 화살표를 사용하여 지속 시간을 조정합니다.
- **포맷팅:** 낮은 확대 수준에서 가사 상자가 너무 작아지면 텍스트를 자동으로 페이드 아웃 및 마스킹하여 인터페이스를 깔끔하게 유지합니다.

## ⚙️ 설정 및 사용자 정의

![설정 대화 상자](docs/images/settings_dialog.png)

재생 바의 기어 아이콘을 통해 설정에 액세스합니다:
- **표시되는 피아노 건반:** 스펙트럼의 피아노 롤에 표시되는 건반 수를 조정합니다.
- **클릭 볼륨:** 메트로놈 세분화 클릭의 음량을 조절합니다.

### 테마
Wavoscope는 완전히 테마를 적용할 수 있습니다. 환경에 맞는 디자인을 선택하세요:
- **Cosmic:** 깊은 보라색과 성운 스타일의 강조.
- **Dark:** 클래식하고 눈이 편안한 다크 모드.
- **Doll:** 에너지 넘치는 핑크와 장난스러운 톤.
- **Hacker:** 검은색 배경에 복고풍 터미널 그린.
- **Light:** 깨끗하고 밝은 전문가용 디자인.
- **Neon:** 일렉트릭 블루와 고대비의 활기찬 색상.
- **OLED:** 최고의 대비를 위한 순수 검정색 배경.
- **Retrowave:** 80년대 신스웨이브 미학.
- **Toy:** 대담한 원색 구성.
- **Warm:** 긴 세션에 적합한 차분하고 편안한 톤.

## 🌍 현지화

Wavoscope는 여러 언어를 지원합니다. **설정 > 글로벌** 탭에서 언어를 변경할 수 있습니다.

### 사용자 지정 번역
Wavoscope는 커뮤니티 주도로 설계되었습니다. `resources/locales` 디렉토리의 JSON 파일을 편집하여 번역을 추가하거나 수정할 수 있습니다.
- 새 언어를 추가하려면 새 JSON 파일(예: `fr.json`)을 생성하고 `"meta": { "name": "Français" }` 필드를 추가합니다.
- 앱은 설정 메뉴에서 유효한 번역 파일을 자동으로 감지하여 목록에 표시합니다.
