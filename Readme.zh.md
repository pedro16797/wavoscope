# Wavoscope - 音频分析与转录工具

Wavoscope 是一款专为音乐人、转录员和音频工程师设计的强大实时音频可视化与转录辅助工具。它提供高保真波形、频谱分析和强大的标记系统，帮助您解析复杂的音频。

![主界面](docs/images/main_view.png)

---

## 🚀 快速入门

### 启动 Wavoscope
Wavoscope 旨在自给自足。您不需要手动安装 Python 或任何其他依赖项。
- **Windows:** 双击 `run.bat`。这将自动设置环境并在根文件夹中创建一个 `Wavoscope.exe` 以供将来使用。
- **Linux/macOS:** 在终端中运行 `bash run.sh`。这将在根文件夹中创建一个 `Wavoscope` 二进制文件。

首次启动时，Wavoscope 将自动下载其自己的 Python 运行时并设置必要的环境。这可能需要几分钟时间，具体取决于您的网络连接。第一次运行后，您可以直接使用生成的 `Wavoscope` 可执行文件（带有应用程序图标）。

### 项目管理与自动保存
Wavoscope 使用“侧边栏”文件系统。当你打开音频文件时，Wavoscope 会在同一目录下创建或加载一个 `.oscope` 文件，用于存储你的标记、循环和设置。
- **打开：** 点击播放栏中的文件夹图标以加载任何常见的音频格式（MP3、WAV、FLAC 等）。
- **保存：** 点击磁盘图标。当有未保存的更改时，该图标会以你主题的强调色闪烁。
- **自动保存：** Wavoscope 会定期自动创建工作快照。你可以在**设置 > 自动保存**选项卡中配置自动保存频率、保留的最大快照数量以及存储位置。默认情况下，仅在有未保存的更改时才会进行自动保存。你可以启用**强制自动保存**，无论是否有更改都始终创建快照。默认情况下，自动保存文件存储在系统的临时文件夹中。

## 🎵 导航与播放

- **缩放：** 在波形或频谱上使用 **鼠标滚轮** 进行放大/缩小。
- **滚动：** 在 **时间轴** 上使用 **鼠标滚轮** 进行前后时间滚动。
- **平移：** **点击并拖动** 波形或频谱以移动时间轴。
- **自适应细分：** 当您缩放时，时间轴会自动调整其网格步长（从 0.01 秒到数小时），确保在不拥挤的情况下提供最佳细节。
- **播放光标：** **左键单击** 波形以移动播放头。
- **速度控制：** 使用底部栏中的滑块调节速度（0.1x 到 2.0x）。Wavoscope 使用高质量的时间拉伸技术，可在保持音高的同时改变速度。
- **速度与打拍测速：** 当前速度（BPM）显示在波形标题栏中。通过连续点击可以手动测量速度（**打拍测速**）。在停止点击 3 秒后，它会自动恢复显示计算出的当前小节速度。

## 🔍 频谱分析与过滤

屏幕下半部分显示常 Q 变换 (CQT) 频谱图，映射到钢琴卷帘。您可以使用频谱分析仪顶部的控件调整 **FFT 窗口** 和 **八度偏移**。

![频谱过滤](docs/images/spectrum_filter.png)

### 高级过滤
您可以使用实时带通滤波器隔离特定的乐器或音符。滤波器手柄（频谱上的垂直线）始终可用：
- **切换截止：** **右键单击** 滤波器手柄以启用或禁用该边界。
- **快速放置：** 在频谱图上的任何位置 **右键单击** 以移动最近的滤波器手柄并启用它。
- **视觉反馈：** 当启用截止时，其范围之外的区域会变暗以帮助您集中注意力。如果两者都禁用，则绕过滤波器。

## 🚩 标记与转录

Wavoscope 使用双标记系统来帮助您规划曲目的结构和和声。

### 节奏标记 (Rhythm/Bar Markers)
- **放置：** 按 `B`（默认）或在时间轴上 **左键单击** 以放置节奏标记。
- **细分：** 打开标记对话框（**右键单击** 标记手柄）以设置细分（例如，四分音符设置为 4）。这些在时间轴上显示为微弱的垂直线。
- **节拍器：** 如果启用了细分点击音，节奏标记会在播放期间自动触发节拍器点击。
- **Shift-Click：** 自动在与前一个标记相同的间隔处放置一个新标记，非常适合快速映射常规节拍。
- **部分：** 将标记设为“部分开始”并为其添加标签（如“主歌”或“副歌”）。

![节奏标记对话框](docs/images/rhythm_dialog.png)

### 和声标记 (Chord Markers)
- **放置：** 按 `C`（默认）或在时间轴上 **右键单击** 以放置和声标记。
- **和弦编辑器：** **右键单击** 现有标记以打开和弦对话框。您可以输入和弦名称（例如 "Am7", "C/G"）或使用选择器。
- **自动分析：** 使用 **建议** 按钮让 Wavoscope 分析该位置的音频并推荐最可能的和弦。
- **试听：** **长按左键** 单击和声标记手柄或点击对话框中的“播放”按钮，通过内置合成器听取和弦。

![和声标记对话框](docs/images/harmony_dialog.png)

### 管理标记
- **拖动：** 您可以 **点击并拖动** 时间轴上的任何标记手柄以微调其位置。
- **重叠：** 当节奏标记和和声标记占据同一空间时，它们会以半高显示（和声在上，节奏在下），以便您仍能与两者交互。
- **循环：** 使用播放栏中的循环按钮在标记之间或整首曲目之间切换。

## 🎤 歌词转录

Wavoscope 具有交互式歌词轨道，可实现高速转录和对齐。

![歌词转录](docs/images/lyrics_track.png)

### 转录工作流
1. **切换轨道：** 点击波形顶部的“歌词”按钮以显示转录轨道。
2. **添加并输入：** 按 `V` 或在歌词轨道的空白处 **单击** 以添加词语。
3. **高速录入：** 在歌词框中输入时，按 **空格** 或 **短横线 (`-`)**。这将自动：
    - 提交当前词语。
    - 紧随其后创建一个新的歌词框（在当前播放头或上一个结束位置）。
    - 将焦点移至新框，以便您在不停止音乐的情况下继续输入。
4. **跳转：** 使用 `Shift + 左/右` 在歌词元素之间跳转。这对于验证时间点非常完美。

### 编辑与调整
- **移动：** **拖动** 歌词框的中心 80% 区域以移动它。
- **时间：** **拖动** 歌词框的边缘（10% 阈值）以调整其开始或结束时间。
- **精度：** 选中歌词时，使用 **方向键** 以 0.1 秒为单位微调。使用 **上/下** 方向键调整持续时间。
- **格式化：** 当歌词框在低缩放级别变得太小时，会自动淡出并遮盖文本，保持界面整洁。

## ⚙️ 设置与自定义

![设置对话框](docs/images/settings_dialog.png)

通过播放栏中的齿轮图标访问设置：
- **可见钢琴键：** 调整频谱钢琴卷帘中显示的键数。
- **点击音量：** 控制节拍器细分点击的音量。

### 主题
Wavoscope 支持完整的主题定制。选择适合您环境的外观：
- **Cosmic:** 深紫色和星云装饰。
- **Dark:** 经典的护眼暗色模式。
- **Doll:** 充满活力的粉色和俏皮色调。
- **Hacker:** 复古终端绿黑配色。
- **Light:** 简洁、高亮度的专业外观。
- **Neon:** 电子蓝和高对比度活力配色。
- **OLED:** 纯黑背景，实现最高对比度。
- **Retrowave:** 80 年代合成器波美学。
- **Toy:** 大胆的三原色。
- **Warm:** 大地色调，适合长时间使用。

## 🌍 本地化

Wavoscope 支持多种语言。您可以在 **设置 > 全局** 选项卡中更改语言。

### 自定义翻译
Wavoscope 旨在由社区驱动。您可以通过编辑 `resources/locales` 目录中的 JSON 文件来添加或修改翻译。
- 要添加新语言，请创建一个新的 JSON 文件（例如 `fr.json`）并添加 `"meta": { "name": "Français" }` 字段。
- 应用程序将自动检测并列出设置菜单中任何有效的翻译文件。
