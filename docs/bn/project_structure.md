# প্রকল্পের গঠন

এই নথিটি Wavoscope রিপোজিটরির ডিরেক্টরি গঠন এবং প্রতিটি উপাদানের উদ্দেশ্য বর্ণনা করে।

## ডিরেক্টরি ওভারভিউ

-   **`src/`**: মূল অ্যাপ্লিকেশনের লজিক এবং সোর্স কোড রয়েছে।
    -   **`audio/`**: মূল অডিও ইঞ্জিন রয়েছে।
        -   `audio_backend.py`: প্রধান অডিও ব্যাকএন্ড ফ্যাসাড (facade)।
        -   `chord_analyzer.py`: কর্ড ফ্ল্যাগের জন্য ক্রোমা-ভিত্তিক কর্ড শনাক্তকরণ।
        -   `ringbuffer.py`: অডিও স্ট্রিমিংয়ের জন্য থ্রেড-সেফ রিং বাফার।
        -   `spectrum_analyzer.py`: FFT এবং স্পেকট্রাল ডেটা গণনার লজিক।
        -   `synth.py`: মেট্রোনোম ক্লিক এবং কর্ড অডিশনিংয়ের জন্য রিয়েল-টাইম সিন্থেসিস।
        -   `waveform_cache.py`: দক্ষ প্রদর্শনের জন্য ওয়েভফর্ম ডেটা তৈরি এবং ক্যাশিং পরিচালনা করে।
    -   **`backend/`**: আধুনিক FastAPI-ভিত্তিক ওয়েব ব্যাকএন্ড।
        -   `main.py`: FastAPI সার্ভারের এন্ট্রি পয়েন্ট, API এন্ডপয়েন্ট এবং ফ্রন্টএন্ড অ্যাসেট সরবরাহ করে।
        -   `state.py`: শেয়ার্ড গ্লোবাল স্টেট (সক্রিয় `Project` ইনস্ট্যান্স)।
        -   `routers/`: অডিও, প্লেব্যাক, প্রজেক্ট ইত্যাদির জন্য মডুলার FastAPI রাউটার।
    -   **`cli/`**: কমান্ড-লাইন ইন্টারফেস ইউটিলিটি রয়েছে।
        -   `flag_cli.py`: টার্মিনালের মাধ্যমে ফ্ল্যাগ ম্যানেজ করার ইউটিলিটি।
        -   `gui.py`: গ্রাফিকাল ইউজার ইন্টারফেসের জন্য `pywebview` লজিক।
        -   `launcher.py`: ব্যাকএন্ড এবং ফ্রন্টএন্ড চালু করার হেল্পার লজিক।
    -   **`frontend/`**: রিঅ্যাক্ট-ভিত্তিক গ্রাফিকাল ইউজার ইন্টারফেস।
        -   `src/components/`: রিঅ্যাক্ট উপাদান (Waveform, Spectrum, Timeline, PlaybackBar)।
        -   `src/store/`: ফ্রন্টএন্ড স্টেট ম্যানেজমেন্ট (Zustand)।
        -   `dist/`: বিল্ট প্রোডাকশন অ্যাসেট।
        -   `tests/`: ফ্রন্টএন্ড উপাদান এবং স্টোর লজিকের ইউনিট টেস্ট।
    -   **`scripts/`**: অটোমেশন এবং ইউটিলিটি স্ক্রিপ্ট (যেমন, স্ক্রিনশট জেনারেশন, লঞ্চার তৈরি)।
    -   **`session/`**: প্রজেক্ট পারসিস্টেন্স এবং হাই-লেভেল স্টেট পরিচালনা করে।
        -   `project.py`: `Project` ক্লাস যা অডিও, মেটাডেটা (ফ্ল্যাগ) এবং ক্যাশিংকে একত্রিত করে।
        -   `manager.py`: `.oscope` সাইডকার ফাইল I/O পরিচালনা করে।
        -   `flags.py`: রিদম এবং কর্ড ফ্ল্যাগ তালিকা পরিচালনা করে।
        -   `undo.py`: ডেল্টা-ভিত্তিক আনডু (undo) ইতিহাস ম্যানেজমেন্ট।
    -   **`tests/`**: ব্যাকএন্ড এবং কোর লজিকের ইউনিট এবং ইন্টিগ্রেশন টেস্ট।
    -   **`utils/`**: সাধারণ সহায়ক ফাংশন এবং শেয়ার্ড ইউটিলিটি (লগিং, কনফিগারেশন ইত্যাদি)।
    -   `main.py`: অ্যাপ্লিকেশনের প্রধান প্রবেশ পয়েন্ট।
    -   `requirements.txt`: পাইথন ডিপেন্ডেন্সি।

-   **`config/`**: অ্যাপ্লিকেশনের কনফিগারেশন ফাইল এবং ডিফল্ট সেটিংস।
-   **`docs/`**: টাস্ক রোডম্যাপ এবং কাঠামোগত নির্দেশিকা সহ প্রকল্পের নথিপত্র।
-   **`resources/`**: আইকন (SVG), থিম (JSON) এবং অ্যাপ্লিকেশন রিসোর্সের মতো স্ট্যাটিক অ্যাসেট।

## রুট ফাইলসমূহ

-   **`run.sh` / `run.bat`**: পরিবেশ সেট আপ এবং অ্যাপ্লিকেশন চালু করার স্ক্রিপ্ট।
-   **`Wavoscope` / `Wavoscope.exe`**: বিল্ট লঞ্চার এক্সিকিউটেবল (স্ক্রিপ্ট দ্বারা উত্পাদিত)।
-   **`AGENTS.md`**: প্রকল্পে কাজ করা এআই এজেন্টদের জন্য নির্দেশিকা এবং রোডম্যাপ।
-   **`CONTRIBUTING.md`**: প্রকল্পে অবদান রাখার জন্য নির্দেশিকা।
-   **`Readme.md`**: সাধারণ প্রকল্পের ওভারভিউ এবং সেটআপ নির্দেশাবলী।
-   **`LICENSE`**: প্রকল্পের MIT লাইসেন্স শর্তাবলী।
-   **`SECURITY.md`**: নিরাপত্তা দুর্বলতা রিপোর্ট করার নীতি।
