# Wavoscope - オーディオ分析・文字起こしツール

Wavoscopeは、ミュージシャン、文字起こし担当者、オーディオエンジニア向けに設計された、強力なリアルタイムオーディオ視覚化および文字起こし支援ツールです。高精細な波形、スペクトラム分析、および堅牢なマーカーシステムを提供し、複雑なオーディオの解析をサポートします。

![メインインターフェース](docs/images/main_view.png)

---

## 🚀 はじめに

### Wavoscopeの起動
Wavoscopeは自己完結型に設計されています。Pythonやその他の依存関係を手動でインストールする必要はありません。
- **Windows:** `run.bat`をダブルクリックします。これにより環境が自動的にセットアップされ、次回以降のためにルートフォルダに`Wavoscope.exe`が作成されます。
- **Linux/macOS:** ターミナルで`bash run.sh`を実行します。これによりルートフォルダに`Wavoscope`バイナリが作成されます。

### 開発とテスト
ソースから実行し、テストを実行する場合：
- **バックエンドテスト:** `PYTHONPATH=src python3 -m pytest src/tests`
- **フロントエンドテスト:** `cd src/frontend && npm test`

初回起動時、Wavoscopeは独自のPythonランタイムを自動的にダウンロードし、必要な環境をセットアップします。インターネット接続状況によっては数分かかる場合があります。初回の実行後は、生成された`Wavoscope`実行ファイル（アプリアイコン付き）を使用するだけで起動できます。

### プロジェクト管理と自動保存
Wavoscopeは「サイドカー」ファイルシステムを使用します。オーディオファイルを開くと、Wavoscopeは同じディレクトリにマーカー、ループ、設定を保存するための`.oscope`ファイルを作成または読み込みます。
- **開く:** 再生バーのフォルダアイコンをクリックして、一般的なオーディオ形式（MP3、WAV、FLACなど）を読み込みます。
- **保存:** フロッピーディスクアイコンをクリックします。未保存の変更がある場合、アイコンはテーマのアクセントカラーで光ります。
- **自動保存:** Wavoscopeは定期的に作業のスナップショットを自動的に作成します。自動保存の頻度、保持するスナップショットの最大数、および保存場所は、**設定 > リカバリ**タブで構成できます。デフォルトでは、未保存の変更がある場合にのみ自動保存が行われます。「強制自動保存」を有効にすると、変更の有無に関わらず常にスナップショットを作成します。デフォルトでは、自動保存はシステムのテンポラリフォルダに保存されます。

## 🎵 ナビゲーションと再生

- **ズーム:** 波形またはスペクトラムの上で**マウスホイール**を使用してズームイン/アウトします。
- **スクロール:** **タイムライン**の上で**マウスホイール**を使用して時間を前後にスクロールします。
- **パン:** 波形またはスペクトラムを**クリックしてドラッグ**し、タイムラインを移動します。
- **適応的分分割:** ズームに合わせてタイムラインのグリッドステップ（0.01秒から数時間まで）が自動的に調整され、過密になることなく最適な詳細レベルが確保されます。
- **再生カーソル:** 波形を**左クリック**して再生ヘッドを移動します。
- **速度制御:** 下部バーのスライダーを使用して、速度を0.1倍から2.0倍まで調整します。Wavoscopeはピッチを維持する高品質なタイムストレッチを使用しています。
- **音量とオーバードライブ：** スライダーで全体の再生音量を調整します。**音量アイコン**をクリックするか、`G` キーを押して**オーバードライブモード**を切り替えます。これにより、音量範囲が 100% から最大 400% まで拡張されます。アプリは、通常モードとオーバードライブモードで個別の音量レベルを記憶します。
- **テンポとタップテンポ:** 現在のテンポ（BPM）が波形ヘッダーに表示されます。繰り返しクリックすることで手動でテンポを測定できます（**タップテンポ**）。3秒間操作がないと、自動的に計算された小節のテンポに戻ります。

## 🔍 スペクトラム分析とフィルタリング

画面の下半分には、ピアノロールに対応した定Q変換（CQT）スペクトログラムが表示されます。スペクトラムアナライザのヘッダーにあるコントロールを使用して、**FFTウィンドウ**と**オクターブシフト**を調整できます。

![スペクトラムフィルタリング](docs/images/spectrum_filter.png)

### 高度なフィルタリング
リアルタイムのバンドパスフィルタを使用して、特定の楽器や音符を分離できます。フィルタハンドル（スペクトラム上の垂直線）は常に利用可能です：
- **カットオフの切り替え:** フィルタハンドルを**右クリック**して、その境界線を有効または無効にします。
- **クイック配置:** スペクトログラム上の任意の場所を**右クリック**して、最も近いフィルタハンドルを移動して有効にします。
- **視覚的フィードバック:** カットオフが有効になると、範囲外の領域が暗くなり、集中しやすくなります。両方が無効な場合、フィルタはバイパスされます。

## 🚩 フラグと文字起こし

Wavoscopeは、トラックの構造と和声をマッピングするためにデュアルフラグシステムを使用します。

### リズムフラグ（リズム/小節マーカー）
- **配置:** `B`（デフォルト）を押すか、タイムラインを**左クリック**してリズムフラグを配置します。
- **分割:** フラグダイアログ（フラグハンドルを**右クリック**）を開き、分割（例：4分音符の場合は4）を設定します。これらはタイムライン上に薄い垂直線として表示されます。
- **メトロノーム:** 分割クリックが有効な場合、リズムフラグは再生中に自動的にメトロノームクリックをトリガーします。
- **Shift-クリック:** 前のフラグと同じ間隔で新しいフラグを自動的に配置します。規則的なビートを素早くマッピングするのに最適です。
- **セクション:** フラグを「セクション開始」としてマークし、ラベル（「Verse」や「Chorus」など）を付けることができます。

![リズムフラグダイアログ](docs/images/rhythm_dialog.png)

### コードフラグ（コードマーカー）
- **配置:** `C`（デフォルト）を押すか、タイムラインを**右クリック**してコードフラグを配置します。
- **コードエディタ:** 既存のフラグを**右クリック**してコードダイアログを開きます。コード名（例：「Am7」、「C/G」）を入力するか、セレクターを使用できます。
- **自動分析:** 「提案」ボタンを使用して、Wavoscopeにその位置のオーディオを分析させ、最も可能性の高いコードを推奨させることができます。
- **試聴:** コードフラグハンドルを**左ボタンで長押し**するか、ダイアログの「再生」ボタンをクリックして、内部シンセサイザーで演奏されるコードを確認できます。

![コードフラグダイアログ](docs/images/harmony_dialog.png)

### フラグの管理
- **ドラッグ:** タイムライン上の任意のフラグハンドルを**クリックしてドラッグ**し、位置を微調整できます。
- **重なり:** リズムフラグとコードフラグが同じ場所にある場合、両方を操作できるように半分ずつの高さ（コードが上、リズムが下）で表示されます。
- **ループ:** 再生バーのループボタンを使用して、マーカー間またはトラック全体のループを切り替えます。

## 🎤 歌詞の文字起こし

Wavoscopeは、高速な文字起こしと配置を可能にするインタラクティブな歌詞トラックを備えています。

![歌詞文字起こし](docs/images/lyrics_track.png)

### 文字起こしワークフロー
1. **トラックの切り替え:** 波形ヘッダーの「歌詞」ボタンをクリックして文字起こしトラックを表示します。
2. **追加と入力:** `V`を押すか、歌詞トラックの空きスペースを**シングルクリック**して単語を追加します。
3. **高速入力:** 歌詞ボックスへの入力中に**スペース**または**ハイフン（`-`）**を押します。これにより以下が自動的に行われます：
    - 現在の単語を確定。
    - その直後に新しい歌詞ボックスを作成（現在の再生ヘッドまたは前の末尾位置）。
    - フォーカスを新しいボックスに移動し、音楽を止めずに入力を続けられます。
4. **シーク:** `Shift + 左/右` または `Shift + A/D` を使用して、歌詞要素間をジャンプします。タイミングの確認に最適です。

### 編集とサイズ変更
- **移動:** 歌詞ボックスの中央80%を**ドラッグ**して移動します。
- **タイミング:** 歌詞ボックスの端（10%の範囲）を**ドラッグ**して、開始時間または終了時間を調整します。
- **精度:** 歌詞が選択されているときに**矢印キー**を使用して0.1秒単位で微調整します。**上/下**矢印で長さを調整します。
- **フォーマット:** ズームレベルが低く歌詞ボックスが小さくなりすぎると、テキストが自動的にフェードアウトおよびマスクされ、インターフェースが整然と保たれます。

## ⚙️ 設定とカスタマイズ

![設定ダイアログ](docs/images/settings_dialog.png)

再生バーのギアアイコンから設定にアクセスします：
- **表示されるピアノキー:** スペクトラムのピアノロールに表示されるキーの数を調整します。
- **クリック音量:** メトロノーム分割クリックの音量を制御します。

### テーマ
Wavoscopeは完全にテーマ変更可能です。環境に合った外観を選択してください：
- **Cosmic:** 深い紫と星雲のようなアクセント。
- **Dark:** クラシックで目に優しいダークモード。
- **Doll:** エネルギッシュなピンクと遊び心のあるトーン。
- **Hacker:** 黒背景にレトロなターミナルグリーン。
- **Light:** 清潔で高輝度なプロフェッショナルな外観。
- **Neon:** エレクトリックブルーと高コントラストな鮮やかさ。
- **OLED:** コントラストを最大化する純黒の背景。
- **Retrowave:** 80年代のシンセウェーブ的な美学。
- **Toy:** 大胆な三原色。
- **Warm:** 長時間のセッションに適した、温かみのある落ち着いたトーン。

## 🌍 ローカライズ

Wavoscopeは複数の言語をサポートしています。「設定 > グローバル」タブで言語を変更できます。

### カスタム翻訳
Wavoscopeはコミュニティ主導で設計されています。`resources/locales`ディレクトリ内のJSONファイルを編集することで、翻訳を追加または変更できます。
- 新しい言語を追加するには、新しいJSONファイル（例：`fr.json`）を作成し、`"meta": { "name": "Français" }`フィールドを追加します。
- アプリは設定メニュー内の有効な翻訳ファイルを自動的に検出してリストします。
